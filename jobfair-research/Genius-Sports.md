## Relevance for Tom (Security/CTO Profile)

**HIGH FIT** — Sports data security is a unique and demanding domain:

1. **Dedicated security roles** — VP Cyber Security, AppSec, K8s Security, DevSecOps
2. **Data integrity is core business** — they have "integrity partnerships" with leagues
3. **Kubernetes-heavy** — container security, pod security policies, cluster hardening
4. **DevSecOps culture** — security integrated into CI/CD pipelines
5. **Ad-tech security** — massive user data processing, privacy concerns
6. **Real-time, low-latency systems** — unique security challenges for live betting data
7. **Regulatory compliance** — sports betting data across multiple jurisdictions
