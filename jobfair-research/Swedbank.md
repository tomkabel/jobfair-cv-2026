## Relevance for Tom (Security/CTO Profile)

**HIGH FIT** — Banking security is one of the most demanding security domains:

1. **Scale of security challenge** — 7.7M customers, one of Europe's largest SIEM platforms
2. **Cloud transformation** — Azure migration creates security architecture opportunities
3. **AI/ML for security** — fraud detection and AML analytics
4. **Regulatory pressure** — PSD2, GDPR, AML, PCI DSS, MIFID2 = organizational mandate for security
5. **Flat hierarchy** — less bureaucracy, more direct impact on security strategy
6. **CTO angle** — largest bank in Estonia needs senior security leadership
