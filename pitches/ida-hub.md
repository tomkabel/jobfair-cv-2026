# IDA Hub — Elevator Pitch (Liftikõne)

**Target:** IDA Hub leadership & ecosystem partners
**Duration:** 45–60 seconds

---

## The Pitch

**[HOOK — 5s]**
"Ida-Virumaa is building a tech ecosystem from the ground up. That's not just an economic opportunity — it's a chance to build security-first infrastructure from the very beginning."

**[WHO I AM — 10s]**
"I'm Tom Kristian Abel — security researcher and CTO. I founded ProksiAbel OÜ doing offensive security consulting, and I'm leading technology at MatX.ee. My team won FELLIN HÄKK 2026. I'm also a student at kood/Jõhvi, which means Ida-Virumaa's tech growth is personal to me."

**[WHY IDA HUB — 15s]**
"IDA Hub is building something rare — a regional tech ecosystem that doesn't just copy Tallinn, but creates its own identity. That identity should include security expertise. Most regional tech hubs struggle to attract security talent because everyone gravitates toward the capital. I'm already here — at kood/Jõhvi — and I want to help build a security culture in Ida-Virumaa's tech community."

**[WHAT I BRING — 15s]**
"I bring enterprise-grade security experience to a regional ecosystem that needs it. Through ProksiAbel OÜ, I've done penetration testing, vulnerability research, and security consulting. My CTO experience at MatX.ee means I understand how to build and lead technical teams. My pet project — a production-grade async API with Docker, Redis, PostgreSQL — shows I can build the kind of infrastructure that growing tech companies need."

**[CLOSING — 5s]**
"IDA Hub is writing the first chapter of Ida-Virumaa's tech story. I want to make sure security is on the first page — because it's much harder to add it in chapter ten."

---

## Delivery Notes

- **"I'm already here"** — physical presence in Ida-Virumaa is your unique advantage
- **Emphasize** regional identity — IDA Hub wants to differentiate from Tallinn, this helps
- **Connect kood/Jõhvi to IDA Hub** — shows you're invested in the local ecosystem
- **On security talent** — honest observation about the industry, shows strategic thinking
- **Final sentence** — "first page vs chapter ten" is a memorable metaphor

## Key Topics to Have Ready

- Ida-Virumaa tech ecosystem development and what security means for growth
- How kood/Jõhvi connects to IDA Hub's mission
- Security talent distribution in Estonia (Tallinn-centric problem)
- What services a regional security consultancy could offer
- Cybersecurity as a differentiator for attracting companies to Ida-Virumaa
