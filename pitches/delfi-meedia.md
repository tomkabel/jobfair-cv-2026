## The Pitch

**[HOOK — 5s]**
"110 million page views a month across 20 portals behind a single paywall — that's not just a media platform, that's a security architecture challenge that most people overlook."

**[WHO I AM — 10s]**
"I'm Tom Kristian Abel — a security researcher and CTO. I founded ProksiAbel OÜ doing offensive security consulting, and I'm currently leading technology at MatX.ee. My team won FELLIN HÄKK 2026 because we think like attackers first, then build defenses."

**[WHY DELFI MEEDIA — 15s]**
"Your micro-frontend architecture is fascinating from a security perspective. Each portal component is a separate application — that means security boundaries between components, API gateway hardening, and session management across the entire Baltic region. Most media companies don't think about this until something goes wrong. You're also integrating AI for translation and data querying — that opens prompt injection and data leakage vectors that need proactive defense."

**[WHAT I BRING — 15s]**
"I've reverse-engineered obfuscated JavaScript architectures and found critical vulnerabilities hiding in the complexity. I understand how paywalls, authentication, and content delivery systems work from the inside — both building and breaking them. My pet project is an async Python API handling media extraction with job queues and expiring download links — similar to how content delivery and paywall systems work at scale."

**[CLOSING — 5s]**
"Delfi reaches a quarter of Estonia's population. I want to make sure the platform that informs a nation is built on a security foundation that's as strong as the journalism it delivers."
