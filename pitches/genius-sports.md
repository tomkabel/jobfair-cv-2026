"240,000 sporting events a year, real-time sub-second data feeds to the NFL — if a single data point is tampered with, the entire integrity of sports betting collapses. That's the security challenge I want to solve."

"Data integrity isn't just a feature for you — it's the entire business. The fact that you have dedicated VP of Cyber Security, Kubernetes security engineers, and DevSecOps roles tells me security is taken seriously, not just audited. Your Tallinn office with 400+ engineers is building real production systems, and the Kubernetes-heavy infrastructure means container security, pod policies, and cluster hardening are daily concerns — exactly the kind of systems I work with."

"I bring deep offensive security experience — I've reverse-engineered complex architectures, bypassed detection systems, and built concurrent tooling for automated vulnerability discovery. My pet project is a production-grade async API with Docker, Redis queues, and PostgreSQL — the same kind of high-throughput data pipeline your sports data feeds depend on. I think about security the way a data integrity officer should: what's the worst case, and how do we prevent it?"

"You're the operating system of modern sport. I want to make sure that operating system is unhackable — because the moment the data can't be trusted, the game is over."

## Delivery Notes

- **The NFL hook is powerful** — everyone knows the brand, it signals scale immediately
- **Pause** after "entire integrity of sports betting collapses" — let the consequence land
- **Emphasize** "data integrity is the entire business" — shows you understand their core value
- **On the Tallinn office** — shows you want to work locally, not remotely
- **Final sentence** — "the game is over" is a double meaning (sports + security) that's memorable

## Key Topics to Have Ready

- Kubernetes security at scale — container isolation, network policies, secrets management
- Real-time data pipeline security — sub-second latency requirements and integrity verification
- Sports betting integrity — how data feeds are protected from manipulation
- The Legend acquisition ($1.2B) — how security scales with M&A
- DevSecOps practices in AWS/Kubernetes environments
- Regulatory compliance across multiple jurisdictions for betting data
